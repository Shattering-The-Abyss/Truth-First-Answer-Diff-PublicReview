# Truth-First Answer Diff Public Review

Public review artifact showing the difference between a polished generic answer and a truth-first answer in a high-risk decision context.

## Boundary

This repo is not financial advice.

The example uses a leveraged ETF question only to demonstrate answer-quality differences:

- avoiding prediction
- avoiding hidden market-state assertions
- preserving user agency
- surfacing risk mechanics
- separating facts from interpretation
